Why is this an issue?

This rule detects the use of discouraged PHP language constructs that should be avoided in Magento 2 code:

  • echo and print: Direct output to the browser is discouraged in Magento 2. Output should be controlled through the proper templating system.
  • Backticks (`): Backtick usage for command execution is a security risk and should be avoided.
function someFunction() {
    echo "Some output";
    print "More output";
    $output = `ls -la`;
}
